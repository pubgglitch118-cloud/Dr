Dr. Ruhm Badr — إعداد Firebase

اتبع الخطوات التالية لتشغيل الموقع بالكامل

  1. أنشئ حساب على firebase.google.com
  2. أنشئ مشروع جديد (مثال: dr-ruhm-badr)
  3. فعّل Authentication ← Sign-in method ← فعّل Email/Password
  4. اذهب إلى Firestore Database ← Create database ← اختر موقع وقواعد اختبارية
  5. اذهب إلى Project Settings ← General ← Your apps ← أضف تطبيق ويب وانسخ الإعدادات
  6. عدّل ملف firebase/config.js بالقيم الحقيقية:
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "dr-ruhm-badr.firebaseapp.com",
  projectId: "dr-ruhm-badr",
  storageBucket: "dr-ruhm-badr.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123",
};
  1. اذهب إلى Authentication ← Users ← Add user ← أضف بريد وكلمة سر للدكتورة
  2. اذهب إلى Firestore Database ← Rules ← الصق القواعد من firebase/rules.firestore
  3. افتح index.html لبدء استقبال الحجوزات
  4. افتح dashboard/index.html للوحة التحكم

📁 هيكل الملفات النهائي:

  index.html             ← صفحة الحجز (خدمات + فورم)
  status.html            ← متابعة الحجز
  setup.html             ← هذه الصفحة
  style.css              ← التصميم الكامل
  app.js                 ← منطق الموقع
  firebase/
    config.js            ← إعدادات Firebase (عدّلها)
    rules.firestore      ← قواعد الأمان
  dashboard/
    index.html           ← لوحة التحكم
    dashboard.js         ← منطق الداشبورد
  PROJECT_MAP.md          ← توثيق كامل
← الموقع الرئيسي ← لوحة التحكم
